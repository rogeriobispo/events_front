# README

## Motivation
  A Simple front end to use with events  api

## Requiriments
    * node 13.0.1
    * react 16.13.1

## Setup
    * git clone git@github.com:rogeriobispo/events_api.git
    * yarn install or npm install
    * yarn start (default port 3000)
